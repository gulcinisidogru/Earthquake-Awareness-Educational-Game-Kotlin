This project aims to raise earthquake awareness among secondary school students. This is a gamified mobile educational application developed within the **Faculty of Technology at Firat University**. The project is built using modern software standards, specifically the **MVVM architecture** and the **Kotlin** language.

## ðŸ“– About the Project
The application presents earthquake preparedness processes, vital actions to take during an earthquake, and post-earthquake emergency management in a fun way that children can understand. It aims for lasting and practical learning by using interactive game modules instead of static information.

For security and privacy reasons, Firebase configuration files have been removed from the project. To run it, you must follow these steps:

* Create your own Android project via **Firebase Console**.

## ðŸŽ® Application Modules

* **Personalization:** A home screen where students can choose their own characters.
* **Educational Quiz:** A dynamic and graded 10-question test system that tests learned information.
* **Interactive Bag:** A drag-and-drop game to identify essential items for an earthquake kit.
* **Stay Calm (Breathing Exercise):** An animation-supported breathing exercise module designed to teach panic management.
